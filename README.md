# Agricultural Region Clustering using K-Means

## 📊 Project Overview
This project aims to group agricultural regions based on environmental conditions and crop yield characteristics using K-Means clustering.

## ⚙️ Techniques Used
- Data Preprocessing
- Feature Scaling (StandardScaler)
- Feature Selection (Correlation Filtering)
- PCA (Dimensionality Reduction)
- K-Means Clustering
- Elbow Method
- Silhouette Score Evaluation

## 📁 Dataset
The dataset contains region-level agricultural data including rainfall, temperature, soil moisture, fertilizer usage, and crop yield.

## 🎯 Objective
To identify similar agricultural regions and classify them into clusters for better analysis and decision-making.

## 📈 Results
- Optimal clusters identified using Elbow Method
- Achieved Silhouette Score ≈ 0.44
- Clusters interpreted based on productivity levels
